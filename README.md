■簡易VJソフト
超シンプルVJソフト
2本の動画を切り替えるだけのソフトです。
VJをやったことない人でも直感的に操作可能です。

■対応形式
基本的にMP4・WebMです
※HTMLのvideoタグで動作する物はOK

■使い方
1.ダウンロードしてきたZIPファイルを任意の場所に解凍し、「controller.html」を開く
2.「スクリーンを開く」を押して新しいタブを開く
3.「簡易VJソフトメイン画面」を見せたいディスプレイ上で「F11」キーで最大化

・左側「動画A用」枠へ、右側「動画B用」枠へ動画ファイルをドラッグ＆ドロップまたは「選択」してください。
・「2秒自動フェード」ボタンやスペースキーを押すとA/Bの映像・音声がなめらかに切り替わります。

※ポップアップブロックが出る場合は解除をお願い致します

■動作環境
ブラウザ（Google Chrome, Firefox, Edge など）

※下記動作確認済み
Windows 11/10
macOS 10.13以降
Google Chrome 144.0.7559.110
Edge 144.0.3719.104
